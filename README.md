# Locativ Homepage (React + Tailwind CSS)

Halaman utama modern untuk layanan dokumentasi event.

## Struktur folder

```bash
.
├── index.html
├── package.json
├── postcss.config.js
├── tailwind.config.js
├── vite.config.js
└── src
    ├── App.jsx
    ├── index.css
    ├── main.jsx
    └── components
        ├── Hero.jsx
        ├── Home.jsx
        └── Navbar.jsx
```

## Menjalankan project

```bash
npm install
npm run dev
```

Buka URL yang ditampilkan Vite (default: `http://localhost:5173`).
